# InteractiveCameraTweak
[![Thunderstore Badge](https://modding.resonite.net/assets/available-on-thunderstore.svg)](https://thunderstore.io/c/resonite/)

This mod fixes an issue where performance drops while using Camera Control.

## Installation

1.  Install this MOD from **[Thunderstore](https://thunderstore.io/c/resonite/p/orange3134/InteractiveCameraTweak/)** using a Mod Manager.
2.  Launch Resonite via your Mod Manager.
